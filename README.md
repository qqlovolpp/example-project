# Example Project

This is an example project created through the GitHub API.

## Description

This repository demonstrates how to use GitHub API tools to programmatically create and manage repositories.

## Features

- Created using GitHub API
- Example C code with utils library
- Makefile for building

## Building

To build the project, simply run:

```
make
```

To clean build artifacts:

```
make clean
```

After building, run the executable with:

```
./example
```
